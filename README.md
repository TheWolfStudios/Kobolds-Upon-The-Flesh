A simple colony sim game I am following a tutorial to make just cause it's been too long since I've done same game dev stuff and want to keep my mind fresh.

File Structure:
	Main - Main Dir Starting Location
		Assets -
			Art - Where art for tiles, entities, etc is stored
				Units - The entity Art
				Tiles - Tile Art
				Building - Building Art
				Items - Item Art
			Audio - Sounds and such
				Units - The entity Art
				Tiles - Tile Art
				Building - Building Art
				Items - Item Art
		Data - Where data resources are stored
			Building - Building Tile Data
			Floor - Floor Tile Data
		Scenes - Where the main scenes in Godot are stored
		Scripts - Where scripts are stored
			Scene - Scene Scripts
			Unit - Unit Scripts
			Item - Item Scripts
			Building - Building Scripts
			Tile - Tile Scripts
			Other - Scripts that do not fit into the other folders
